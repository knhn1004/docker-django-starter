# docker-django-starter

## using django-3-jet

## shell scripts

- create super user

```bash
./create_super.sh
```

- install new python package

```bash
./install.sh (package_name)
```

- uninstall python package

```bash
./uninstall.sh (package_name)
```

- rename the django project

```bash
./rename_project.sh (old_name) (new_name)
```
